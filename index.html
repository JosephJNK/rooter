Rooter

Tiny cross-platform hashchange/routing for the browser

Download as .zip Download as .tar.gz View on GitHub

Information

Package rooter
Description Tiny cross-platform hashchange/routing

Usage

Route

// Create routes
rooter.route('/', function(){
  console.log("index");
});
rooter.route('/test/:id', function(args){
  console.log(args.id);
});

//Intiailize
rooter.init();

Hash

rooter.hash.listen(function(hash){
  //Do something with the new hash
});

//Change the hash
rooter.hash.value('/test/20');

//Get the current hash
rooter.hash.value();

//Manually trigger hash event
rooter.hash.trigger('/test/10');
